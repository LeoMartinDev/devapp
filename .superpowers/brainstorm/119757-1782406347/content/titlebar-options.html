Quel style de barre de titre ?

La barre de titre native affichera le path + les infos git. Le reste de l'UI (sidebar, contenu) ne change pas.

A

Crochets

Branche entre crochets, séparateur tiret cadratin —

dev/devapp [main] — devapp
Sidebar
Logs / Terminal
B

Parenthèses

Branche entre parenthèses, plus discret

dev/devapp (main) — devapp
Sidebar
Logs / Terminal
C

Épure

Dossiers/chemin puis tiret, branche en texte libre

dev/devapp — main — devapp
Sidebar
Logs / Terminal
D

Séparateur pipe

Séparation nette entre chemin, branche, et app

dev/devapp | main | devapp
Sidebar
Logs / Terminal
